Описание файлов проекта:
1. install.bat - скрипт установки всех необходимых плагинов в папке для запуска бота.
2. add_admin.py - скрипт создания зашифрованного файла с данными админов и операторов и добавлением первого админа. 
3. main.py - монолит с кодом всего бота. 
4. .env - API Id b и API HASH тг приложения, через которое бот делает запросы к ТГ.

Инструкция по запуску бота: 
1. Запускаем файл install.bat 
2. Открываем в папке бота Powershell.
3. Выполняем команду $env:BOT_TOKEN="ТОКЕН ТГ БОТА"
4. Выполняем команду .venv\Scripts\activate.bat
5. Выполняем команду python add_admin.py "ТГ ID первого админа".
6. Выполняем команду python main.py "ТОКЕН ТГ БОТА"

После можно работать с ботом, запуская его через команду /start или соответствующую кнопку. 

Для админа доступны команды "Добавить администратора", "Добавить оператора", "Удалить оператора", "Список операторов", "Сканировать". 

Для оператора доступна команда "Сканировать". 

На каждый номер телефона формируется сессия, которая действует с момента авторизации по этому телефону 15 минут, после чего авторизацию нужно проходить заново.

После сканирования каждого чата необходимо заново вводить номер телефона, по которому необходимо новое сканирование.
